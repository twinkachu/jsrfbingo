Setting up multiplayer for Xbox

Note: these instructions are only for original xbox. If you're using CXBX, follow Auddy's tutorial video.

For this to work, you need the EvolutionX dashboard (build 3935) and an internet connection to your xbox. Set TSR to debug in the settings. Download all of the necessary files (you’ll need an xbe and the evox bridge index/bat files), and transfer the xbe over to your console. You may also need to rename the xbe to whatever the default JSRF xbe is called so that you can launch it from the games menu instead of the file browser. Update the index.js file with your xbox’s IP address (from the evox dashboard).

To start the mod: launch the game, then open the bridge (the bat file), then open the website and click listen (if you’re hosting the session) or connect (if you’re connecting to an existing session). The “not connected” text on your in-game start screen should update to say “ready”. Set your tick rate to 6 - if you hit your white button, you should see send/acknowledge/receive indexes - the numbers should be changing.
